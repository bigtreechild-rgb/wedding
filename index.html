⚙️ 一次性設定

請先填入你的 Firebase 設定

即時連線(掃碼、同步答題、排行榜)需要一個免費的 Firebase 即時資料庫。設定一次就好。

  1. 到 console.firebase.google.com 建立一個專案(免費)。
  2. 左側選單 →「Realtime Database」→ 建立資料庫 → 測試模式啟動。
  3. 專案設定 →「你的應用程式」→ 新增「網頁應用程式」
  4. 複製 firebaseConfig 內容,貼到這個 HTML 檔最上方的 FIREBASE_CONFIG 區塊,存檔。
  5. 把這個檔案上傳到任何網頁空間(GitHub Pages / Firebase Hosting / Netlify 皆可),用該網址掃碼即可。

⚠️ 婚禮當天為求方便,Realtime Database 規則設為公開讀寫即可(遊戲結束後可關閉)。要我幫你產生規則就跟我說。

🪑 席位卡名單

賓客名單(每人一張專屬 QR)

每行一位賓客,格式「姓名,桌號」。賓客掃自己座位上的 QR,系統就知道他是誰、坐哪桌,免輸入暱稱。

桌號可以是數字或文字(如「主桌」「親友桌」)。留空桌號也可以,只是不參加桌次競賽。

🎵 精選點歌歌單
